I've been working with Virtual Reality since 2015, focusing on VR development and UX design. I've given talks about building VR experiences with web technologies, and in 2016 I challenged myself to create a new VR project every day for a month.

I'm interested in what's possible with VR on the web, and I continue to experiment with new approaches to immersive experiences.

{% include global/wide-h3.html text="Current Setup" %} {% include posts/figure.html src="max.jpg" class="pull-right circle" %}

I'm currently using an Oculus Quest tethered to my PC for running VR experiences. Not much has changed since I last wrote about the Quest.
