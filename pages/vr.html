--- title: VR Stuff permalink: /vr/ description: VR experiments, talks, and projects built with web technologies since 2015. ---

I've been building VR experiences since 2015, back when WebVR was still finding its feet. I've given talks on developing immersive web experiences, and in 2016 I set myself a challenge: create a new VR project every day for a month. I actually did it.

I'm interested in what's possible when you combine VR with web technologies. The accessibility, the immediacy, the weird experimental stuff you can't do anywhere else. This is where I document the experiments, talks, and projects that came out of that.
